EnvironmentQuery2D
==================

**Inherits:** ``Node2D``

A node used to gather environment data using a :ref:`QueryGenerator2D<class_QueryGenerator2D>` and evaluating them using :ref:`QueryTest2D<class_QueryTest2D>`\ s.

.. rst-class:: classref-introduction-group

Description
-----------

**EnvironmentQuery2D** provides a flexible, node-based system for AI agents to ask spatial questions about the environment. Queries are composed by adding one :ref:`QueryGenerator2D<class_QueryGenerator2D>` as child of the environment query and assigning :ref:`QueryTest2D<class_QueryTest2D>`\ s as children of the :ref:`QueryGenerator2D<class_QueryGenerator2D>`.

Queries are time sliced in the background to maintain performance.
